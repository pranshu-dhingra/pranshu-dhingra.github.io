# pranshu-dhingra.github.io

This repository hosts my personal portfolio site, powered by GitHub Pages.

## Overview

This is a central hub that links to my project demos and the code behind them. The site is built using simple Markdown files and requires no external build tooling.

## Structure

- `index.md` - Main landing page with introduction
- `projects.md` - List of all projects with links to demos and code

## Adding a New Project

To add a new project to the portfolio:

1. Edit `projects.md`
2. Copy the template block at the bottom of the file
3. Replace the placeholders:
   - `<Project Title>` - Your project name
   - `<One-line description...>` - Brief description of what it does and why it matters
   - `<repo-name>` - Your project repository name
4. Commit and push your changes

## GitHub Pages Configuration

This site is published via GitHub Pages with the following settings:
- **Source**: Deploy from main branch, root directory (`/`)
- **URL**: https://pranshu-dhingra.github.io/

To enable GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages section
3. Set Source to "Deploy from a branch"
4. Select branch: `main`, folder: `/ (root)`
5. Save
